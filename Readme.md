
### Install

```bash
npm i onebook/react-util
```

### APIs

```js
import { allRefVals, refVal, focus, clear } from 'react-util'
```

* `this::allRefVals()`
* get val - `this::refVal('some')` or set val - `this::refVal('some', 'value')`
* `this::focus('some')`
* `this::clear('some')`
