
### Install

```bash
npm i onebook/react-util
```

### APIs

```js
import {
  allRefVals,
  refVal,
  focus,
  clear
} from 'react-util'
```

* `this::allRefVals()`
* `this::refVal('ref')`
  - get val - `this::refVal('ref')`
  - set val - `this::refVal('ref', 'value')`
* `this::focus('ref')`
* `this::clear('ref')`
